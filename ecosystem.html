Python Basics¶

Either when learning Python with no prior programming experience or migrating to Python from other programming languages (Matlab, Perl, etc.) one of the first concepts to learn is how Python is set up. The purpose of this session is to provide a high level introduction to Python to give you a general feel for the capability of this language.
